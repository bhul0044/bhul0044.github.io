5 HTML tags i have learnt

p tag
paragraph tag is a tag we use when we have to add long texts or texts in general.
h1 tag
h1 tag is used to give heading but it can be used only once in a particular page or project.
br tag
br tag is used to add single line breaks in between lines.
title tag
title tag is used to denote title to the document.
blockquote tag
blockquote tag is used define quote from another books or sources
